📱 Tentang Aplikasi Ini

CatSpace Planner adalah aplikasi perencana teritori kucing yang membantu pemilik kucing untuk:

  • Mendata jumlah kucing dan karakteristik masing-masing
  • Menyesuaikan tata letak rumah dengan kebutuhan kucing
  • Menganalisis tingkat konflik antar kucing
  • Memvisualisasikan pergerakan kucing selama 24 jam
  • Memberikan rekomendasi jumlah tempat makan dan kotak pasir

🧮 Konsep Numerik & Algoritma (Untuk Pengembangan Fungsional)

1. Perhitungan Tingkat Konflik

Rumus Dasar:

            ConflictScore(cat_i, cat_j) = BaseScore(relation) + TraitInfluence(cat_i) + TraitInfluence(cat_j)
            
            Dimana:
            - BaseScore(relation):
              • Best Friends    → 10
              • Roommates       → 30  
              • Tidak Cocok     → 80
            
            - TraitInfluence(cat):
              • Dominance Influence  = (Dominance - 50) / 5
              • Tolerance Influence  = (100 - Tolerance) / 5
              • Total Trait Influence = DominanceInfluence + ToleranceInfluence
              • (Hanya berlaku untuk kucing dengan status "Baru")
            
            Total Conflict = Σ ConflictScore(cat_i, cat_j) / jumlah pasangan

Interpretasi: Semakin tinggi nilai, semakin besar potensi konflik.

2. Rekomendasi Jumlah Peralatan

Formula:

            Tempat Makan Minimal   = ⌈ jumlahKucing × 1.3 ⌉
            Kotak Pasir Minimal    = jumlahKucing + 1
            
            Rumus ini berdasarkan:
            - Golden Rule of Cat Nutrition: 1 tempat makan per kucing + 30% cadangan
            - Cat Litter Box Rule: 1 kotak pasir per kucing + 1 tambahan

3. Skala Pergerakan dengan Luas Rumah

Konversi Luas ke Pixel:

            Luas Rumah (m²)        = user input
            Sisi Rumah (meter)     = √luasRumah
            Skala (pixel/meter)    = (canvasSize × 0.85) / sisiRumah
            Offset (pixel)         = (canvasSize - (sisiRumah × skala)) / 2
            
            Posisi Kucing (pixel)  = offset + (posisiMeter × skala)

Kecepatan Gerak: Tetap 2.5 pixel/frame (tidak tergantung luas rumah), sehingga pergerakan selalu nyaman dilihat.

4. Algoritma Deteksi Konflik & Animasi

State Machine untuk Perkelahian:

            IF jarak(kucing_i, kucing_j) < thresholdKonflik AND hubungan = "Tidak Cocok" THEN
                statusFighting = true untuk kedua kucing
                fightTimer = 40 frame (≈ 1.3 detik)
                
                // Efek tolakan
                posisiBaru = posisiLama + (arah × force)
                
                // Perubahan arah
                arahBaru = arahLama + random offset
                arahBaru = normalize(arahBaru)
            END IF

5. Algoritma Pergerakan Kucing

Brownian Motion dengan Bouncing:

            Setiap frame:
                posisi.x += arah.x × kecepatan
                posisi.y += arah.y × kecepatan
                
                IF posisi menyentuh batas rumah THEN
                    arah.x = -arah.x atau arah.y = -arah.y
                    posisi = batas + margin
                END IF
                
                IF random() < 0.02 THEN // 2% chance per frame
                    arah.x += (random() - 0.5) × 0.8
                    arah.y += (random() - 0.5) × 0.8
                    arah = normalize(arah) // panjang vektor = 1
                END IF

Kecepatan Dinamis:

            kecepatanDasar = 2.5 pixel/frame
            IF status = "Baru" THEN
                kecepatan = 2.5 + (stress / 100)  // Kucing stres bergerak lebih cepat
            END IF
            IF fighting = true THEN
                kecepatan = 4.0  // Lari saat berantem
            END IF

6. Visualisasi dengan Canvas

Teknik Rendering:

  • Grid: Menggambar garis setiap 1 meter sesuai skala
  • Kucing: Lingkaran dengan warna unik per kucing + fitur wajah (mata, hidung, telinga)
  • Garis Hubungan:
    • Garis merah putus-putus = konflik aktif
    • Garis merah solid = hubungan tidak cocok
    • Garis hijau = best friends
    • Garis pink = roommates
  • Efek Perkelahian: Lingkaran merah berdenyut dan tanda seru (!) di atas kucing

📌 Catatan untuk Pengembangan Fungsional

Saat ini aplikasi masih dalam tahap mockup/interaktif dengan simulasi data. Untuk pengembangan menjadi aplikasi yang fully functional, diperlukan:

  • Backend Database: Menyimpan data kucing, preferensi, dan riwayat konflik
  • User Authentication: Multi-user profile untuk menyimpan berbagai skenario rumah
  • Machine Learning: Untuk memprediksi pola konflik berdasarkan karakteristik kucing
  • Export Feature: Mengexport rekomendasi ke PDF atau berbagi dengan veterinarian
  • IoT Integration: Terhubung dengan smart feeder dan smart litter box untuk monitoring real-time
