Description

This computation application mockup simulates territorial behavior among multiple cats in a household. Each cat is modeled as an agent with behavioral traits.

How to Use

  • Enter the number of cats
  • Provide personality traits
  • Set house parameters
  • Click "Run Simulation"

Numerical Concepts & Algorithms

  • Agent-Based Modeling (ABM)
  • Monte Carlo Simulation
  • Nonlinear interaction model
  • Time-stepping iteration
